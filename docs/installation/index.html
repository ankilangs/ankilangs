        Installation#

        Installing Anki#

        If you don’t have Anki yet, check the official instructions for installing on Windows, macOS, Linux, Android, or iOS.

        Installing a Deck#

        1. Download the .apkg file from the deck’s page (list of all decks)
        2. Open Anki
        3. Go to File → Import
        4. Select the downloaded .apkg file
        5. Click “Import”

        The deck will be added to your collection and you can start studying immediately.

        Updating a Deck#

        When a new version is released:

        1. Download the new .apkg file
        2. Import it into Anki (same steps as above)
        3. Anki will update your deck while preserving your learning progress

        Your review history and card scheduling will be maintained during updates.
